## üìö Academic Publications

### HoaLibrary Original Paper

**"HOA Library, A Set of C++ Classes for High Order Ambisonics"**  
*Pierre Guillot, Eliott Paris, Julien Colafrancesco*  
ICMC-SMC 2014 - Joint Conference, Athens, Greece

**Abstract**: The HoaLibrary is a free and open-source set of C++ classes for High Order Ambisonics (HOA) designed for real-time applications. It provides tools for encoding, decoding, rotating, and performing various transformations on 2D and 3D ambisonic sound fields. The library has been implemented across multiple platforms including Max, Pure Data, and OpenFrameworks.

**Key Topics**:
- Mathematical foundations of HOA
- Implementation architecture
- Real-time performance optimization
- Cross-platform design considerations
- Applications in spatial audio

**Download**: [ICMC_SMC_2014_Proceedings-HOA.pdf](http://hoalibrary.mshparisnord.fr/wp-content/uploads/2014/12/ICMC_SMC_2014_Proceedings-HOA.pdf)

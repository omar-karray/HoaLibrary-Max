The HoaLibrary (high order ambisonics library) is a two and three dimensional high order 
ambisonics processing set of externals and patches for Pure Data, Max, Faust and VST. 
It allows you to synthesize, transform and modulate sound fields.

It has been elaborated to facilitate the use of the circular and spherical harmonics and 
the routing in the ambisonic domain for musicians and composers.  
Thus, they should be able to let their creativity run free.

------------------------------------------------------------------------------------------

INSTALLATION:

 - Remove any old version of HoaLibrary from your Packages folder
 - Copy the HoaLibrary folder to: ~/Documents/Max 9/Packages/
 - Restart Max if it's already running
 - Test by creating objects: hoa.2d.encoder~, hoa.3d.decoder~, hoa.map

UNINSTALLATION:

 - Remove the HoaLibrary folder from ~/Documents/Max 9/Packages/
